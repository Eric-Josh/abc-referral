## Abc Referral

## Installation

- Clone Repository
    > `git clone https://github.com/Eric-Josh/abc-referral`

- Install all dependencies
    > `cd abc-referral`

    > `composer install or composer update`

    > `npm install`

- Create DB

- Copy .env.example to .env
    > `cp .env.example .env`

- Generate APP_KEY
    > `php artisan key:generate`

- Configure your DB and mail server

- Run Migration
    > `php artisan migrate`

- Run app
    > `php artisan serve`
